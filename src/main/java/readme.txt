Консольная "змейка".
SnakeSection - кусочек змеи.
SnakeDirection - класс (enum) будет описывать направление движения змеи.
Mouse - мышь, не двигается (статическая) с координатами в "комнате".
Snake - собственно, змейка. У змеи есть состояние (жива/мертва), голова (самый первый кусочек), направление
движения и список секций.
Room - комната с размерами width и height), змеей и мышью.
